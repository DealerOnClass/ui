Purpose

This website serves as a reference for the DealerOn UI Kit based on the Bootstrap(v4) Framework.

It is intended to be used for the design and development of DealerOn web entities.

It should EXTEND Bootstrap and be compatible with existing sites on older versions of Bootstrap.
